# nodeny-mono-term
Модуль взаємодії терміналів Monobank з білінговою системою для інтернет провайдерів Nodeny версій 49 (50)

## Встановлення

- Скопіювати mono.pl в папку /usr/local/www/apache22/cgi-bin/
- прописати дозволені ip в  .htaccess
- створити файл /usr/local/nodeny/mono-term.log с правами для запису для веб-сервера
- вставить платіжну систему в файл /usr/local/nodeny/web/paystype.pl :
    96 => 'Платіжний термінал Monobank',

В якості платіжного кода використовується код, відображений у кожного абонента в його кабінеті внизу ("Ваш персональный платежный код: …")
